## 파일 공유 설정 (중요!)

각 파일을 앱에서 사용하려면 **반드시** 공유 설정을 해야 합니다:

1. 구글 드라이브에서 파일 선택
2. 우클릭 → "공유" 클릭
3. "일반 액세스" → **"링크가 있는 모든 사용자"** 선택
4. "뷰어" 권한으로 설정
5. "링크 복사" 클릭

## 앱에 추가하는 방법

### 1단계: 관리자 로그인
1. 앱 로그인 화면에서 타이틀을 5번 탭
2. "스님(관리자) 로그인" 버튼이 나타남
3. 암호 입력: `108`
4. Google 계정으로 로그인

### 2단계: 콘텐츠 추가
1. 대시보드에서 "법문 콘텐츠 관리" 클릭
2. "콘텐츠 추가" 버튼 클릭
3. 미디어 타입 선택:
   - **YouTube**: 유튜브 링크
   - **구글 드라이브**: 드라이브 파일 (영상, 오디오, PDF, 텍스트)
   - **텍스트**: 직접 법문 내용 작성

### 3단계: 구글 드라이브 파일 등록
1. "구글 드라이브" 탭 선택
2. 파일 종류 선택:
   - 영상 파일 (.mp4, .mov 등)
   - 오디오 파일 (.mp3, .wav 등)
   - PDF 문서
   - 텍스트 파일
3. 복사한 공유 링크 붙여넣기
4. 제목 입력 (예: "반야심경 해설")
5. 법사명 입력 (예: "정수스님")
6. 길이 입력 (선택사항, 예: "15:30")
7. "등록" 클릭

## 파일 링크 형식

구글 드라이브 공유 링크는 다음과 같은 형식입니다:
```
https://drive.google.com/file/d/[파일ID]/view?usp=sharing
```

또는
```
https://drive.google.com/open?id=[파일ID]
```

## 문제 해결

### 파일이 재생되지 않아요
- 공유 설정을 "링크가 있는 모든 사용자"로 했는지 확인
- 링크를 정확히 복사했는지 확인

### 썸네일이 표시되지 않아요
- 영상 파일의 경우 구글 드라이브가 자동으로 썸네일 생성
- 오디오/PDF는 기본 아이콘 표시

### 폴더 전체를 한번에 추가하고 싶어요
- 현재는 보안상 파일을 하나씩 추가하는 방식
- 필요시 개발자에게 요청하여 일괄 추가 기능 구현 가능

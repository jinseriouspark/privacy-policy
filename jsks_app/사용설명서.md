## 앱 개요

### 앱 목적
정수결사는 법륜사 신도님들의 매일 수행을 돕고, 절의 일정과 법문을 공유하는 수행 관리 앱입니다.

### 주요 기능
- ✅ 매일 수행 체크리스트 기록
- 📅 절 행사 일정 확인 및 참석 신청
- 🎧 오늘의 법문 영상/음성 시청
- 📊 나의 수행 기록 통계 확인
- 👤 법명 설정 및 프로필 관리

### 접속 방법

## 일반 신도용 화면

### 1. 로그인 화면

#### 기능
- Google 계정으로 간편 로그인
- 앱 소개 문구 표시

#### 사용 방법
1. 앱을 실행하면 로그인 화면이 나타납니다
2. **"Google 계정으로 시작하기"** 버튼을 클릭
3. Google 계정을 선택하여 로그인
4. 최초 로그인 시 수행 목표 선택 화면으로 이동

#### 주의사항
- 반드시 Google 계정이 필요합니다
- 로그인 정보는 안전하게 보관됩니다

---

### 2. 수행 목표 선택 (온보딩)

#### 기능
- 매일 실천할 수행 항목 선택
- 나만의 수행 체크리스트 구성

#### 사용 방법
1. 최초 로그인 후 자동으로 나타나는 화면입니다
2. 제시된 수행 항목 중 실천하고 싶은 항목을 선택
   - 예: "오전 예불", "경전 읽기", "108배", "자비 명상" 등
3. 여러 개 선택 가능 (최소 1개 이상)
4. **"선택 완료"** 버튼 클릭
5. 홈 화면으로 이동

#### 나중에 변경하기
- **계정 > 수행 목표 선택** 메뉴에서 언제든지 수정 가능

### 3. 홈 화면 (메인)

#### 화면 구성
```
┌─────────────────────────┐
│ 평안하신가요            │
│ [법명/이름]님           │
├─────────────────────────┤
│ [이번 주 수행 캘린더]   │  ← 월~일 진행 상황
├─────────────────────────┤
│ [오늘의 수행 점검]      │  ← 수행 체크 카드
│ [오늘의 법문]           │  ← 법문 보기 카드
├─────────────────────────┤
│ [다가오는 일정]         │  ← 최대 3개 표시
└─────────────────────────┘
```

#### 1) 주간 캘린더
- **기능**: 이번 주(월~일) 수행 완료 상황 확인
- **표시**:
  - 오늘: 검은색 배경
  - 완료된 날: 초록색 점 표시
  - 미완료: 회색 점 또는 빈 상태
- **클릭**: 일정 탭으로 이동

#### 2) 오늘의 수행 점검 카드
- **표시**: "N/M 완료" (완료한 항목 수 / 전체 목표 수)
- **색상**:
  - 미완료: 흰색 배경 + 초록색 테두리
  - 완료: 초록색 배경 + 흰색 글자 + 축하 효과
- **클릭**: 수행 점검 화면으로 이동

#### 3) 오늘의 법문 카드
- **표시**: 주황색 배경 카드
- **내용**: "법문을 들어보세요"
- **클릭**: 법문 목록 화면으로 이동

#### 4) 다가오는 일정
- **표시**: 오늘 이후 일정 최대 3개
- **정보**: 날짜, 시간, 제목
- **클릭**: 일정 상세 정보 팝업

### 4. 오늘의 수행 점검 화면

#### 기능
- 오늘 실천한 수행 항목 체크
- 실시간 진행률 확인
- 수행 기록 저장

#### 화면 구성
```
┌─────────────────────────┐
│ ← 오늘의 수행 점검      │
│ [날짜]                  │
├─────────────────────────┤
│ [진행바] N 완료 / M 목표│
├─────────────────────────┤
│ ☑ 오전 예불             │
│ □ 경전 읽기             │
│ ☑ 108배                 │
│ □ 자비 명상             │
│ ...                     │
├─────────────────────────┤
│ [수행 완료하기 🎉]      │ ← 100% 달성 시
│ [나중에 이어서 하기]    │ ← 미완료 시
└─────────────────────────┘
```

#### 사용 방법
1. 각 수행 항목을 클릭하여 체크/해제
2. 체크된 항목:
   - 초록색 배경으로 변경
   - 체크 아이콘 표시
3. 상단 진행바에서 현재 진행률 확인
4. 완료 버튼 클릭:
   - **100% 완료**: "수행 완료하기 🎉" 버튼 (초록색)
   - **미완료**: "나중에 이어서 하기" 버튼 (회색)
5. 저장 완료 후 홈 화면으로 복귀

#### 주의사항
- 하루에 한 번만 기록 가능
- 저장 후에도 다시 수정 가능
- 자정이 지나면 새로운 날짜로 초기화

### 5. 일정 화면

#### 기능
- 월간/주간 캘린더로 일정 확인
- 절 공식 일정 및 개인 일정 관리
- 행사 참석 신청

#### 화면 구성
```
┌─────────────────────────┐
│ 일정          [주간/월간]│
├─────────────────────────┤
│ ◀ 2025년 1월 ▶          │
├─────────────────────────┤
│ 월 화 수 목 금 토 일     │
│ 1  2  3  4  5  6  7     │ ← 일정 있는 날 초록색 표시
│ 8  9  10 11 12 13 14    │
│ ...                     │
├─────────────────────────┤
│ 1월 15일                │
├─────────────────────────┤
│ [일정 카드 1]           │
│ [일정 카드 2]           │
├─────────────────────────┤
│ [일정 추가하기]         │
└─────────────────────────┘
```

#### 사용 방법

**1. 캘린더 보기**
- **월간 보기**: 전체 월 캘린더
- **주간 보기**: 이번 주만 표시
- 상단 토글 버튼으로 전환
- 좌우 화살표로 이전/다음 월 이동

**2. 날짜 선택**
- 캘린더에서 날짜 클릭
- 하단에 해당 날짜 일정 표시
- 일정 있는 날은 숫자 하단에 "N개" 표시

**3. 일정 상세 보기**
- 일정 카드 클릭
- 팝업으로 상세 정보 표시:
  - 제목
  - 날짜 및 시간
  - 장소 (있는 경우)
  - 첨부파일 (있는 경우)
  - 참석 인원 현황

**4. 참석 신청**
- 일정 상세 팝업에서 **"참석 신청하기"** 버튼 클릭
- 이미 신청한 경우: **"참석 취소하기"** 표시
- 인원 제한이 있는 경우: "N/M명 신청" 표시
- 마감된 경우: "마감되었습니다" 표시

**5. 개인 일정 추가**
- 하단 **"일정 추가하기"** 버튼 클릭
- 제목, 날짜, 시간 입력
- 필요시 첨부파일 추가 (Google Drive)
- **"등록"** 버튼 클릭
- 본인에게만 보이는 개인 일정

#### 일정 종류
- 🏛️ **절 공식 일정**: 스님이 등록한 행사 (모두 볼 수 있음)
- 📝 **개인 일정**: 본인만 볼 수 있는 일정

### 6. 오늘의 법문 화면

#### 기능
- 스님이 올린 법문 영상/음성/자료 시청
- 카테고리별 분류
- 법문 검색

#### 화면 구성
```
┌─────────────────────────┐
│ 오늘의 법문             │
│ 지혜의 말씀을...        │
├─────────────────────────┤
│ [전체] [경전공부]       │
│ [참선법회] [공부자료]   │ ← 필터 버튼
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │ [썸네일 이미지]     │ │
│ │ [경전공부] 30:15    │ │ ← 카테고리 + 시간
│ └─────────────────────┘ │
│ 반야심경 해설           │
│ 지월스님                │
├─────────────────────────┤
│ (더 많은 법문 카드...)  │
└─────────────────────────┘
```

#### 사용 방법

**1. 카테고리 필터**
- **전체**: 모든 법문 표시
- **경전공부**: 경전 강의 및 해설
- **참선법회**: 참선 수행 관련
- **공부자료**: 교재 및 참고 자료
- 상단 버튼 클릭으로 선택

**2. 법문 보기**
- 법문 카드 클릭
- 영상 타입에 따라:
  - **YouTube**: YouTube 앱/브라우저로 열림
  - **Drive 영상**: Google Drive 미리보기
  - **Drive 음성**: 음성 재생
  - **PDF**: PDF 뷰어로 열림

**3. 카드 정보**
- 썸네일 이미지
- 카테고리 태그 (상단 왼쪽)
- 재생 시간 (상단 오른쪽)
- 제목
- 법사명
- 재생 버튼 아이콘

#### 주의사항
- 인터넷 연결 필요
- 영상은 데이터가 많이 소모될 수 있음
- WiFi 환경 권장

### 7. 나의 수행 기록 화면

#### 기능
- 수행 기록 통계 확인
- 연속 수행 일수 확인
- 항목별 수행 횟수 분석

#### 화면 구성
```
┌─────────────────────────┐
│ 나의 수행 기록          │
├─────────────────────────┤
│ ┌─────────┬───────────┐ │
│ │ 이번달  │ 연속수행  │ │
│ │ 15회 📊 │ 7일 🔥   │ │
│ └─────────┴───────────┘ │
├─────────────────────────┤
│ [주간] [월간] [통계]    │ ← 보기 모드
├─────────────────────────┤
│ 이번주 수행 현황        │
│ ┌─────────────────────┐ │
│ │   월 화 수 목 금 토 일│ │
│ │ 예불 ●○●●●○○        │ │
│ │ 경전 ●●○●●○○        │ │
│ │ 108배○●●●○○○        │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

#### 보기 모드

**1. 주간 히트맵 (기본)**
- 이번 주(월~일) 수행 현황
- 각 수행 항목별 완료 여부를 점으로 표시
- ● 초록색 점: 완료
- ○ 회색 점: 미완료

**2. 월간 캘린더**
- 이번 달 전체 수행 기록
- 날짜별 수행 강도를 색상으로 표시
- 색이 진할수록 많이 완료
- 100% 완료한 날은 별 표시

**3. 통계 차트**
- 이번 달 항목별 수행 횟수
- 막대 그래프로 표시
- 가장 많이 한 수행 순서대로 정렬

#### 상단 통계 카드
- **이번 달**: 이번 달 전체 수행 횟수
- **연속 수행**: 연속으로 수행한 날짜 (스트릭)
  - 하루라도 빠지면 0으로 초기화
  - 오늘 수행하면 1 증가

#### 사용 방법
1. 상단 통계로 한눈에 파악
2. 보기 모드 전환 (주간/월간/통계)
3. 각 항목 클릭 시 상세 정보 확인 (구현 예정)

### 8. 계정 (프로필) 화면

#### 기능
- 법명 설정 및 수정
- 수행 목표 재설정
- 참석 신청한 일정 확인
- 앱 설정 및 로그아웃

#### 화면 구성
```
┌─────────────────────────┐
│ [프로필사진] 홍길동     │
│              email@...  │
├─────────────────────────┤
│ 7일 연속수행 | 5개 목표 │
├─────────────────────────┤
│ 나의 법명               │
│ 정진행 [✏️]            │
├─────────────────────────┤
│ ☑️ 수행 목표 선택       │
│ 📋 수행 기록 보기 방식  │
│ 📅 참석 신청한 일정     │
│ 🔔 알림 설정            │
│ 🚪 로그아웃             │
└─────────────────────────┘
```

#### 사용 방법

**1. 법명 설정/수정**
- "나의 법명" 섹션의 ✏️ 버튼 클릭
- 법명 입력
- ✓ 버튼으로 저장
- 저장 중 "1초의 휴식..." 화면 표시
- 홈 화면에서 법명으로 표시됨

**2. 수행 목표 선택**
- 메뉴 클릭
- 온보딩 화면으로 이동
- 수행 항목 재선택
- 선택 완료 후 프로필로 복귀

**3. 수행 기록 보기 방식**
- 메뉴 클릭
- 팝업에서 원하는 방식 선택:
  - 주간 히트맵 (기본)
  - 월간 캘린더
  - 타임라인 목록
  - 통계 차트
- 선택 시 즉시 적용
- "나의 수행 기록" 화면에 반영됨

**4. 참석 신청한 일정**
- 메뉴 클릭
- 내가 참석 신청한 모든 일정 목록 표시
- 일정 클릭 시 상세 정보 및 취소 가능

**5. 알림 설정**
- 메뉴 클릭
- 알림 권한 설정
- 수행 리마인더 시간 설정 (구현 예정)

**6. 로그아웃**
- 메뉴 클릭
- 로그인 화면으로 이동
- 로그인 정보는 유지됨 (재로그인 간편)

```
┌─────────────────────────┐

│    업로드→검토→게시     │
```

### 1. 오늘의법문 자료관리

#### 기능
- 법문 영상/음성/자료 업로드
- 게시 전 검토 (초안/게시 상태 관리)
- 카테고리 분류
- 게시된 법문 관리

#### 화면 구성
```
┌─────────────────────────┐
│ ← 오늘의법문 자료관리   │
├─────────────────────────┤
│ [콘텐츠 추가] [YouTube] │
├─────────────────────────┤
│ 대기중 3개 | 게시됨 12개│
├─────────────────────────┤
│ ● 대기 중인 콘텐츠 (3)  │
│ ┌─────────────────────┐ │
│ │ [썸네일] 반야심경... │ │
│ │ [✓법문으로게시][삭제]│ │
│ └─────────────────────┘ │
├─────────────────────────┤
│ ● 게시된 법문 (12)      │
│ ┌─────────────────────┐ │
│ │ [썸네일] 금강경...   │ │
│ │ [비공개전환][삭제]   │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

#### 사용 방법

**1. 법문 업로드 (드라이브 파일)**
1. **[콘텐츠 추가]** 버튼 클릭
2. 업로드 폼 작성:
   - **파일 선택**: "드라이브에서 파일 선택" 클릭
     - Google Drive 팝업 열림
     - 스님 전용 폴더에서 파일 선택
   - **제목**: 법문 제목 입력 (필수)
   - **법사명**: 법사명 입력 (기본값: 지월스님)
   - **설명**: 법문 설명 (선택사항)
   - **카테고리**: 4개 중 선택 (필수)
     - 전체
     - 경전공부
     - 참선법회
     - 공부자료
3. **[등록]** 버튼 클릭
4. "콘텐츠가 등록되었습니다" 메시지
5. 자동으로 자료관리 화면으로 이동
6. "대기 중" 섹션에 표시됨

**2. 법문 업로드 (YouTube)**
1. **[YouTube]** 버튼 클릭
2. 업로드 폼 작성:
   - **YouTube 링크**: URL 붙여넣기 (필수)
     - 예: https://www.youtube.com/watch?v=xxxxx
   - **제목**: 법문 제목 입력 (필수)
   - **법사명**: 법사명 입력
   - **설명**: 법문 설명 (선택사항)
   - **카테고리**: 4개 중 선택 (필수)
3. **[등록]** 버튼 클릭
4. YouTube 썸네일 자동 추출
5. "대기 중" 섹션에 추가됨

**3. 법문 검토 및 게시**
- **대기 중인 콘텐츠**:
  - 업로드 직후 상태
  - 신도에게 보이지 않음
  - 스님만 확인 가능
- **검토하기**:
  - 썸네일, 제목, 법사명 확인
  - 내용 미리보기 (클릭 시 재생)
- **게시하기**:
  1. **[✓ 법문으로 게시]** 버튼 클릭
  2. 확인 팝업: "신도들에게 공개됩니다"
  3. 확인 클릭
  4. "게시된 법문" 섹션으로 이동
  5. 신도 화면에 즉시 표시됨

**4. 게시된 법문 관리**
- **비공개 전환**:
  1. 게시된 법문의 **[비공개로 전환]** 클릭
  2. 확인 팝업
  3. "대기 중" 섹션으로 이동
  4. 신도 화면에서 숨겨짐
- **삭제**:
  1. **[삭제]** 버튼 클릭
  2. 확인 팝업: "삭제하시겠습니까?"
  3. 확인 클릭
  4. 영구 삭제됨

#### 주의사항
- 게시 후에도 언제든 비공개 전환 가능
- 삭제는 되돌릴 수 없음
- YouTube 링크는 유효한 URL만 가능

### 2. 회원 수행 현황

#### 기능
- 신도들의 수행 기록 모니터링
- 날짜별/주간/월간 통계
- 회원별 수행 순위
- 오름차순/내림차순 정렬

#### 화면 구성
```
┌─────────────────────────┐
│ ← 회원 수행 현황        │
├─────────────────────────┤
│ [날짜별][주간][월간]    │
├─────────────────────────┤
│ 정렬 [높은순↓][낮은순↑]│
├─────────────────────────┤
│ [조회 날짜: 2025-01-15] │ ← 날짜별만
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │ 홍길동              │ │
│ │ hong@...       85% │ │
│ │ [예불][경전][108배] │ │
│ └─────────────────────┘ │
│ ┌─────────────────────┐ │
│ │ 이순신              │ │
│ │ lee@...        100%│ │
│ │ [예불][경전][108배] │ │
│ │ [명상]              │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

#### 사용 방법

**1. 보기 모드 선택**
- **📅 날짜별**:
  - 특정 날짜의 모든 회원 수행 현황
  - 각 회원의 완료한 수행 항목 표시
  - 진행률(%) 표시
- **📊 주간**:
  - 이번 주(월~오늘) 회원별 수행 횟수
  - 횟수로 순위 표시
- **📈 월간**:
  - 이번 달(1일~오늘) 회원별 수행 횟수
  - 누적 횟수로 순위 표시

**2. 정렬 방식**
- **높은순 ↓ (내림차순)**:
  - 날짜별: 진행률 높은 순 (100% → 0%)
  - 주간/월간: 수행 횟수 많은 순
- **낮은순 ↑ (오름차순)**:
  - 날짜별: 진행률 낮은 순 (0% → 100%)
  - 주간/월간: 수행 횟수 적은 순

**3. 날짜 선택 (날짜별 모드)**
- 날짜 입력 필드 클릭
- 달력에서 날짜 선택
- 해당 날짜 데이터 자동 로드

**4. 데이터 해석**

**날짜별 예시:**
```
홍길동
hong@example.com    85%
[예불] [경전읽기] [108배]
```
- 해석: 홍길동님이 4개 목표 중 3개 완료 (85%)
- 미완료 항목: 명상 (표시되지 않음)

**주간/월간 예시:**
```
이순신
lee@example.com     15회
```
- 해석: 이순신님이 이번 주/달에 총 15회 수행 기록

#### 활용 방법
- 꾸준히 수행하는 신도 파악
- 수행이 부족한 신도에게 격려
- 법회에서 모범 사례 공유
- 수행 통계를 법회 자료로 활용

### 3. 일정 관리

#### 기능
- 절 공식 행사 등록
- 참석 인원 제한 설정
- 첨부파일 업로드 (안내문 등)
- 참가자 명단 확인

#### 화면 구성
```
┌─────────────────────────┐
│ ← 일정 관리             │
├─────────────────────────┤
│ [+ 행사 등록]           │
├─────────────────────────┤
│ 등록된 행사             │
│ ┌─────────────────────┐ │
│ │ 정기법회            │ │
│ │ 2025-01-20 14:00    │ │
│ │ 절 공식 일정        │ │
│ │                     │ │
│ │ 참석 인원           │ │
│ │ 25 / 30명          │ │
│ │ [등록한유저확인]    │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

#### 사용 방법

**1. 행사 등록**
1. **[+ 행사 등록]** 버튼 클릭
2. 등록 폼 작성:
   - **일정 제목**: 행사 이름 입력 (필수)
     - 예: "정기법회", "초하루법회", "참선수련"
   - **날짜**: 달력에서 선택 (필수)
   - **시간**: 시간 선택 (필수)
   - **참석 인원 설정**:
     - **무제한**: 제한 없음 (기본)
     - **인원 제한**: 최대 인원수 입력
       - 예: 30명
       - 마감 시 신도가 신청 불가
   - **첨부파일** (선택사항):
     - **드라이브에서 파일 선택** 클릭
     - 행사 안내문, 교재 등 첨부
     - 또는 URL 직접 입력
3. **[일정 등록]** 버튼 클릭
4. "일정이 등록되었습니다" 메시지
5. 등록된 행사 목록에 추가
6. 신도 화면에 즉시 표시

**2. 참가자 명단 확인**
1. 등록된 행사 카드에서 **[등록한 유저 확인하기]** 클릭
2. 팝업으로 참가자 목록 표시:
   - 법명 (또는 이름)
   - 이메일 주소
   - 신청 순서
3. 명단 출력 가능 (브라우저 인쇄 기능)
4. **[닫기]** 버튼으로 복귀

**3. 참석 인원 현황**
- 실시간 인원 표시:
  - 무제한: "15명 신청"
  - 제한 있음: "25 / 30명" (진행바 포함)
- 마감 임박 시 자동 알림 (구현 예정)

**4. 행사 수정/삭제**
- 현재는 스님이 직접 Supabase에서 수정 필요
- 향후 업데이트 예정

#### 인원 제한 예시

**무제한 설정:**
```
참석 인원
32명
```
- 제한 없이 누구나 신청 가능

**인원 제한 설정 (30명):**
```
참석 인원
28 / 30명
[■■■■■■■■■□]
```
- 28명 신청, 2자리 남음
- 30명 도달 시 "마감" 표시

### 4. 앱 문구 설정

#### 기능
- 로그인 화면 문구 수정
- 홈 화면 인사말 수정
- 법문 화면 설명 수정

```
┌─────────────────────────┐
│ ← 앱 설정 관리          │
├─────────────────────────┤
│ 로그인 화면             │
│ ┌─────────────────────┐ │
│ │ 앱 이름(타이틀)     │ │
│ │ [정수결사         ] │ │
│ │                     │ │
│ │ 앱 설명(부제)       │ │

│ └─────────────────────┘ │
├─────────────────────────┤
│ 홈 화면                 │
│ ┌─────────────────────┐ │
│ │ 인사말 뒷부분       │ │

│ └─────────────────────┘ │
├─────────────────────────┤
│ 법문 & 시스템           │
│ ┌─────────────────────┐ │

│ │                     │ │

│ │                     │ │

├─────────────────────────┤

```

#### 사용 방법

**1. 문구 수정**
1. 수정하고 싶은 입력 필드 클릭
2. 기존 텍스트 수정 또는 새로 입력
3. 각 섹션별 수정 가능:
   - **로그인 화면**:
     - 앱 이름: 상단 큰 제목 (기본: "정수결사")
     - 앱 설명: 부제목 (기본: "매일의 수행을...")
   - **홈 화면**:
     - 인사말: "평안하신가요" + [법명]님
   - **법문 화면**:
     - 제목: 화면 최상단 (기본: "오늘의 법문")
     - 설명: 제목 하단 설명 (기본: "지혜의 말씀을...")
   - **시스템**:
     - 로딩 메시지: 법명 저장 시 표시

**2. 설정 저장**
1. 모든 수정 완료 후 하단 **[설정 저장하기]** 클릭
2. Supabase에 저장
3. "설정이 저장되었습니다. 페이지를 새로고침합니다" 메시지
4. 자동 새로고침
5. 신도 화면에 즉시 반영

**3. 미리보기**
- 저장 전 변경 사항 미리보기 불가
- 저장 후 신도 계정으로 로그인하여 확인

#### 문구 작성 가이드
- **간결하게**: 50-60대가 쉽게 이해
- **따뜻하게**: 친근하고 격려하는 톤
- **명확하게**: 기능이 분명히 드러나도록

## 부록: 자주 묻는 질문 (FAQ)

### Q1. 로그인이 안 돼요
**A**: Google 계정으로만 로그인 가능합니다. Chrome 브라우저 사용을 권장합니다.

### Q2. 수행 기록이 저장 안 돼요
**A**: 인터넷 연결을 확인해주세요. 저장 버튼을 꼭 눌러야 기록됩니다.

### Q3. 법명을 어떻게 바꾸나요?
**A**: 계정 > 나의 법명 > ✏️ 버튼 클릭 > 수정 > ✓ 저장

### Q4. 참석 신청을 취소하고 싶어요
**A**: 일정 상세 화면에서 "참석 취소하기" 버튼을 누르세요.

### Q5. 법문 영상이 안 나와요
**A**: 인터넷 연결을 확인하고, YouTube 앱이 설치되어 있는지 확인하세요.

### Q6. 수행 목표를 변경하고 싶어요
**A**: 계정 > 수행 목표 선택 > 항목 재선택 > 선택 완료

### Q7. 알림을 받고 싶어요
**A**: 계정 > 알림 설정에서 권한을 허용해주세요. (일부 기능 개발 중)

### Q8. 앱이 느려요
**A**: 브라우저 캐시를 삭제하거나 앱을 재시작해보세요.

### Q9. 스님께 질문하고 싶어요
**A**: 절 공식 일정 중 "질의응답" 시간을 활용하거나, 직접 연락주세요.

### Q10. 내 정보를 삭제하고 싶어요
**A**: 스님께 요청하시면 계정을 삭제해드립니다.

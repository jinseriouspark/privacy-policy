# 버전 관리 가이드

## 캐시 강제 갱신 방법

사용자의 브라우저 캐시를 강제로 지우고 최신 버전으로 업데이트하려면:

1. `index.html` 파일을 엽니다
2. 35-84번 줄 사이의 `APP_VERSION` 값을 변경합니다
   ```javascript
   const APP_VERSION = '1.0.1'; // 이 값을 변경하세요 (예: '1.0.2')
   ```
3. 변경 후 커밋하고 배포합니다

## 버전 변경 시점

다음과 같은 경우 버전을 올려주세요:

- ✅ 주요 기능 추가
- ✅ UI/UX 대대적 변경
- ✅ 데이터 구조 변경
- ✅ 버그 수정 (중요한 경우)
- ✅ 캐시 문제로 사용자가 이전 버전을 보고 있는 경우

## 버전 규칙

- **메이저 버전** (1.x.x): 대규모 기능 추가 또는 구조 변경
- **마이너 버전** (x.1.x): 작은 기능 추가
- **패치 버전** (x.x.1): 버그 수정

## 자동으로 지워지는 것들

버전이 변경되면 다음이 자동으로 지워집니다:
- localStorage (사용자 세션과 수행 항목은 유지)
- sessionStorage
- Cache Storage
- Service Workers

## 주의사항

- 사용자 로그인 정보는 유지됩니다
- 수행 항목 캐시는 유지됩니다
- 버전 변경 시 자동으로 페이지가 새로고침됩니다
